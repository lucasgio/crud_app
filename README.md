
# CRUD APP Productos

Aplicación para crear productos con estructura de CRUD. Esta desarrollado con React
React Router, Redux




## Autor

- [Iosvany Alvarez](https://www.github.com/lucasgio)


## Instalación

Instalar CRUDApp con npm

```bash
  git clone https://github.com/lucasgio/crud_app.git
  cd crud_app
  npm install
  npm start
```
